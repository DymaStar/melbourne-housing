# Прогнозування цін на житло в Мельбурні

## Опис проєкту

Цей проєкт демонструє повний цикл роботи Data Analyst та Machine
Learning над задачею прогнозування вартості житла на основі датасету
**Melbourne Housing Snapshot**.

Мета проєкту --- побудувати модель регресії, яка прогнозує ціну
нерухомості за її характеристиками.

------------------------------------------------------------------------

## Датасет

**Назва:** Melbourne Housing Snapshot

**Цільова змінна:** `Price`

Датасет містить інформацію про житлові об'єкти: - кількість кімнат; -
тип житла; - площу ділянки; - площу будинку; - відстань до центру
міста; - рік побудови; - район; - дату продажу; - координати.

------------------------------------------------------------------------

## Етапи роботи

1.  Завантаження даних.
2.  Первинний аналіз даних.
3.  Очищення даних.
4.  Дослідницький аналіз (EDA).
5.  Створення нових ознак (Feature Engineering).
6.  Підготовка даних до моделювання.
7.  Навчання моделей машинного навчання.
8.  Оцінювання моделей.
9.  Прогнозування вартості житла.
10. Формування висновків.
