# Melbourne Housing Price Prediction

## Project Overview

This project demonstrates the complete workflow of a Data Analyst and Machine Learning project using the **Melbourne Housing Snapshot** dataset.

The objective is to predict house prices based on property characteristics using regression models.

The project covers the entire machine learning pipeline, from loading raw data to evaluating predictive models.

---

## Dataset

**Dataset:** Melbourne Housing Snapshot

**Target variable:** `Price`

The dataset contains information about residential properties in Melbourne, including:

* Number of rooms
* Property type
* Distance from city centre
* Land size
* Building area
* Year built
* Region
* Sale date
* Geographic coordinates
* Other property characteristics

---

## Project Workflow

1. Data loading

5. Feature Engineering

8. Model evaluation

## Machine Learning Models

The project compares several regression approaches:

* Dummy Regressor (Baseline)
* Linear Regression
* Random Forest Regressor

---

## Evaluation Metrics

Model performance is evaluated using:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
